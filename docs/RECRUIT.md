# 「計算とは何か」を、自分の手で動かして確かめてみないか?

### — WHILE 言語インタプリタ・プロジェクト 参加者募集(3年生歓迎)

## 30行で書けて、チューリング完全

このプロジェクトで扱う **WHILE 言語**には、代入と while ループしかありません。データは `nil` と `cons` で作る二分木だけ。整数すらありません。

```
read X;
  Y := nil;
  while X do {
    Y := cons (hd X) Y;
    X := tl X;
  }
write Y
```

それでもこの言語は**チューリング完全** — 原理的にはどんなプログラムでも書けます。「計算可能性」の授業で出てくるチューリング機械は、テープと状態遷移で頭の中だけで動かすものでした。WHILE は違います。**文法定義も意味論も全部読める小ささなのに、本物のプログラミング言語として動く**。停止性問題も、万能プログラム(自己解釈系)も、講義の証明の中ではなく、自分のターミナルの中で再現できます。

## 教科書がそのまま「実行」できる

この言語は Neil D. Jones の名著 *Computability and Complexity: From a Programming Perspective* (The MIT Press) で定義されたものです。この本の野心は「**計算可能性理論と計算量理論を、チューリング機械ではなくプログラミング言語で再構築する**」こと。本リポジトリはその WHILE 言語の処理系の実装です:

- **OCaml 製インタプリタ** — コア言語の意味論(教科書の Definition 2.2.2–2.2.4)をほぼ逐語的に実装
- **Haskell 製脱糖器** — `if`・`case`・パターンマッチ・プロシージャなどの拡張構文を、コア言語へ変換するプログラム変換器

教科書のページ番号がソースコードのコメントに書いてある、という珍しいコードベースです。「本を読む」と「コードを書く」が直結します。

## DIKU で生まれた研究の系譜につながる

Jones はコペンハーゲン大学(DIKU)で TOPPS グループを率い、ここから現代のプログラミング言語研究の重要なアイデアがいくつも生まれました。WHILE 言語はその共通の土台です:

- **部分評価と二村射影** — 「インタプリタを特化するとコンパイラが自動的に得られる」という魔法のような定理。世界初の自己適用可能な部分評価器 (mix) は DIKU で WHILE 系の言語に対して実現されました
- **プログラミング言語による計算量クラスの特徴付け** — 「`cons` を禁止した WHILE プログラムが計算できるのはちょうど PTIME」のように、言語の制限が計算量クラスに対応するという美しい結果(教科書後半のテーマ)
- **サイズ変化停止性解析、プログラム変換、自己解釈系** — そして近年は、WHILE を土台にした**可逆プログラミング言語(R-WHILE)**の研究へも発展しています

つまりこのプロジェクトは「小さな教育用処理系いじり」に見えて、**国際的な研究の入口**に立っています。

## やってみたいことが選べます

テスト基盤(hspec / OUnit2、カバレッジ 100%)を整備済みなので、安心して手を入れられる状態です。例えば:

| 興味 | テーマ例 |
|---|---|
| 言語処理系を作りたい | 新しい構文糖衣の追加(`or`、`let`、パターンの拡張)、エラーメッセージ改善 |
| 理論を動かしたい | **WHILE で書いた WHILE の自己解釈系**(教科書の万能プログラム)を動かす |
| 研究の入口に立ちたい | WHILE の**部分評価器**を書いて二村射影を実験する/可逆版 WHILE への拡張 |
| Web が好き | ブラウザで WHILE を書いて実行できるプレイグラウンドの改良 |

使う技術は **Haskell・OCaml・パーサジェネレータ(BNFC)** — 関数型言語と言語処理系の実践的スキルがそのまま身につきます。コードベースは小さく(全体で 1000 行程度)、3年生の今の知識で全体を見渡せるサイズです。

**「プログラムとは何か」を、プログラムを書いて考える。** 興味が湧いたら、まず `git clone` して reverse.while を動かすところから始めましょう。
